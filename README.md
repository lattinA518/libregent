## 🖥️ What libregent does

libregent is a C++20 library with Python bindings for text simplification. It is built for RegenT-style processing and works with text files in CoNLL-U format. If you want to explore how a sentence can be made easier to read, libregent gives you the tools to do that in a structured way.

This project fits tasks like:

- simplifying complex English text
- working with linguistic data
- handling natural language processing tasks
- reading and writing CoNLL-U files
- using the library from C++ or Python
